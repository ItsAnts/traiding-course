# Trading Course Materials Repository

## Описание

Данный репозиторий содержит материалы из уроков курса по трейдингу.

Внутри размещаются:

- Доски с занятий  
- Изображения и схемы  
- Дополнительные материалы  
- Полезные ссылки  
- Практические примеры  
- Вспомогательный софт  

Репозиторий используется как централизованное хранилище всех учебных материалов.

---

## Структура репозитория

Материалы разделены по урокам.

Каждый урок хранится в отдельной папке.

project-root/

├── lesson_1/
│   ├── boards/        # Доски с занятия
│   ├── images/        # Изображения и схемы
│   ├── materials/     # Текстовые материалы
│   └── links.md       # Полезные ссылки

├── lesson_2/
│   ├── boards/
│   ├── images/
│   ├── materials/
│   └── links.md

├── lesson_3/
│   └── ...

├── utilities/         # Вспомогательный софт
│   ├── platforms/     # Торговые платформы
│   ├── tools/         # Вспомогательные утилиты
│   └── guides/        # Инструкции по установке

└── README.md


⸻

Назначение папок

lesson_1, lesson_2, lesson_3 …

Папки уроков содержат материалы конкретного занятия.

Внутри могут находиться:
	•	boards/ — доски, записи объяснений
	•	images/ — изображения и схемы
	•	materials/ — текстовые файлы, конспекты
	•	links.md — список полезных ссылок

Каждый урок имеет собственную папку.

Название формируется по шаблону:

lesson_1  
lesson_2  
lesson_3  


⸻

utilities/

Папка содержит вспомогательный софт и инструменты.

Используется для хранения:
	•	Торговых платформ
	•	Вспомогательных программ
	•	Скриптов
	•	Инструкций по установке

Возможные подпапки:

utilities/

├── platforms/     # Торговые платформы
├── tools/         # Вспомогательные программы
├── scripts/       # Скрипты
└── guides/        # Инструкции


⸻

Правила добавления материалов
	1.	Материалы добавляются только в папку соответствующего урока.
	2.	Названия файлов должны быть понятными.
	3.	Изображения размещаются в папке images/.
	4.	Записи с досок размещаются в папке boards/.
	5.	Ссылки добавляются в файл links.md.
	6.	Софт размещается только в папке utilities/.

⸻

Форматы файлов

Допустимые форматы:
	•	.png
	•	.jpg
	•	.pdf
	•	.md
	•	.txt
	•	.pptx
	•	.zip

Допускаются и другие форматы при необходимости.

⸻

Назначение репозитория

Репозиторий используется для:
	•	Хранения учебных материалов
	•	Быстрого доступа к материалам уроков
	•	Централизации информации
	•	Упрощения повторения пройденных тем
	•	Распространения обновлений материалов

⸻

Обновление материалов

Материалы могут обновляться:
	•	После каждого урока
	•	При добавлении новых схем
	•	При добавлении новых полезных ссылок
	•	При обновлении программного обеспечения

Рекомендуется регулярно проверять наличие обновлений.
