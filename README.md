# Trading Course Materials Repository

## Описание

Данный репозиторий содержит материалы из уроков курса по трейдингу.

Внутри размещаются:

- Доски с занятий  
- Изображения и схемы  
- Дополнительные материалы  
- Полезные ссылки  
- Практические примеры  
- Вспомогательный софт  

Репозиторий используется как централизованное хранилище всех учебных материалов.

---

## Структура репозитория

Материалы разделены по урокам.

Каждый урок хранится в отдельной папке.


Обновление материалов

Материалы могут обновляться:
	•	После каждого урока
	•	При добавлении новых схем
	•	При добавлении новых полезных ссылок
	•	При обновлении программного обеспечения

Рекомендуется регулярно проверять наличие обновлений.
